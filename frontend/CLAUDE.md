# Asmika Fashion UI Direction

Asmika Fashion is NOT a SaaS dashboard or generic luxury template.

The brand direction is:
- editorial
- cinematic
- couture
- emotionally luxurious
- fashion-magazine inspired
- restrained and confident

The website should feel closer to:
- Saint Laurent campaigns
- Jacquemus editorials
- Loewe
- Dior
- Vogue layouts
- luxury perfume websites

NOT:
- startup landing pages
- generic dark UI
- Webflow luxury clones
- over-animated premium templates

UI Philosophy:
- image-first storytelling
- asymmetrical composition
- whitespace
- restrained typography
- cinematic atmosphere
- premium spacing rhythm
- emotional visual storytelling

Typography:
- elegant
- smaller scale
- refined hierarchy
- avoid oversized centered hero text

Color Direction:
- ivory
- champagne
- warm beige
- mocha
- charcoal
- muted bronze
- soft whites

Avoid:
- harsh pure black
- loud gradients
- oversaturated colors
- aggressive gold
- neon accents

Motion:
- subtle
- slow
- cinematic
- smooth
- restrained

The site should feel like a luxury fashion editorial experience, not a premium React template.
